# Netflix-Data-Analysis-Project
End-to-end data analysis on Netflix dataset using Python, Pandas, and Visualization.
Project Overview

This project analyzes Netflix movies and TV shows dataset to extract insights about content distribution, trends, and audience targeting.

Business Problem

Analyze Netflix content to understand:

Content growth over time
Audience targeting
Country-wise production
Content type distribution
🛠 Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Key Insights
Movies dominate Netflix content
Content increased rapidly after 2015
USA produces most content
TV-MA and TV-14 ratings dominate
Peak content addition: 2017–2019
Drama & Comedy are top genres
Visualizations
Histogram
KDE Plot
Box Plot
Violin Plot
Scatter Plot
Pie Chart
Heatmap
Conclusion

Netflix has expanded rapidly with a strong focus on movies and adult content, with opportunities to grow in international and family-friendly content.

Netflix Movies and TV Shows Dataset (Kaggle)
